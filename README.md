# Archangelix

Simple console-based exam system.

## Usage

```bash
python3 archangelix.py show_exams
```
List and run an interactive exam.

```bash
python3 archangelix.py show_results
```
Show saved exam results.

## Structure

- `exams/` - Exam files in JSON format
- `results/` - Automatically exam result files in JSON format
